csvbase (alpha)

csvbase is a website for hosting table data.

It's primarily aimed at open data — but private data is accepted too.

  • To get started, just upload your CSV file or paste directly from MS Excel
  • All tables have a simple REST API
    • A GraphQL API will follow soon
  • Exporting to Pandas/R/MS Excel is dead easy; just use the url:
    • import pandas as pd; pd.read_csv("{{url_for('csvbase.table_view', username='meripaterson', table_name='stock-exchanges', _external=True)}}")

Some possible uses might be:

  • Hosting open data, for example the world's stock exchanges
  • A simple back-end for basic applications
  • Sharing data between developers (who like APIs!) and data analysts (who like CSVs!)

csvbase is open source software, under the AGPLv3. It's (mostly) written by Cal Paterson. Please let me know what you think by sending an email.
